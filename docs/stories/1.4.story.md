### Implementation Approach

**Shadcnblocks Reference Global CSS:**

- Source file: `Shadcnblocks References/globals.css`
- Contains comprehensive CSS setup with:
  - Multiple Google Font imports (Inter, Roboto, Poppins, etc.)
  - Tailwind CSS v4 imports using @import "tailwindcss"
  - @tailwindcss/typography plugin integration
  - CSS custom properties for theming (light and dark modes)
  - Animation keyframes for various effects
  - Font family variables using --font-sans and --font-mono

**Font Variable Integration:**

- Keep layout.tsx unchanged with Geist font imports and variables
- Current layout.tsx defines: `--font-geist-sans` and `--font-geist-mono`
- Shadcnblocks globals.css expects: `--font-sans` and `--font-mono`
- Solution: Update the @theme inline section in globals.css to map:
  - `--font-sans: var(--font-geist-sans);`
  - `--font-mono: var(--font-geist-mono);`
- This preserves existing Geist fonts while using the new CSS structure
